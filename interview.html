IDisposable exposes Dispose for releasing unmanaged resources. 'using' ensures Dispose is called even on exceptions. Use when wrapping streams, DB connections, etc.

Covariance allows a more derived type than originally specified (out T), contravariance allows a less derived type (in T). Used in delegates and generic interfaces for safe type substitutions.
