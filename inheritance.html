Inheritance in C#

Learn how to create class hierarchies and inherit functionality

Theory

Inheritance is one of the fundamental principles of object-oriented programming (OOP). It allows a class (derived/child class) to inherit properties, methods, and other members from another class (base/parent class).

Key Concepts:
  • Base Class (Parent): The class whose members are inherited
  • Derived Class (Child): The class that inherits from the base class
  • Single Inheritance: C# supports single inheritance only (one base class per derived class)
  • Inheritance Chain: Classes can form inheritance hierarchies
  • Access Modifiers: Control what can be inherited (public, protected, private)

Example Code

Basic Inheritance Example
using System;

// Base class (Parent)
public class Vehicle
{
    protected string brand;
    protected int year;
    
    public Vehicle(string brand, int year)
    {
        this.brand = brand;
        this.year = year;
    }
    
    public virtual void Start()
    {
        Console.WriteLine($"{brand} vehicle started");
    }
    
    public void DisplayInfo()
    {
        Console.WriteLine($"Brand: {brand}, Year: {year}");
    }
}

// Derived class (Child)
public class Car : Vehicle
{
    private int doors;
    
    public Car(string brand, int year, int doors) : base(brand, year)
    {
        this.doors = doors;
    }
    
    // Override base class method
    public override void Start()
    {
        Console.WriteLine($"{brand} car engine started with key");
    }
    
    // New method specific to Car
    public void OpenTrunk()
    {
        Console.WriteLine($"{brand} car trunk opened");
    }
    
    public void DisplayCarInfo()
    {
        DisplayInfo(); // Inherited method
        Console.WriteLine($"Doors: {doors}");
    }
}

// Usage example
class Program
{
    static void Main()
    {
        Car car = new Car("Toyota", 2023, 4);
        
        car.Start();           // Output: Toyota car engine started with key
        car.DisplayCarInfo();  // Uses inherited DisplayInfo()
        car.OpenTrunk();       // Car-specific method
    }
}
