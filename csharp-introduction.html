Introduction to C# & .NET

Learn the fundamentals of C# programming language and the .NET framework

What is C#?

Understanding C# and its place in modern development

C# (pronounced "C-Sharp") is a modern, object-oriented programming language developed by Microsoft. It runs on the .NET framework and is designed for building Windows applications, web applications, and more.

Key Features:
  • Type Safety: Strong type checking prevents runtime errors
  • Memory Management: Automatic garbage collection
  • Object-Oriented: Supports inheritance, polymorphism, and encapsulation
  • Cross-Platform: Runs on Windows, macOS, and Linux with .NET Core
  • Rich Library: Extensive Base Class Library (BCL)
  • Modern Language: Regular updates with new features

Basic C# Program Structure

Understanding the anatomy of a C# program

Hello World Example

Every C# program follows a basic structure. Here's a simple "Hello World" program:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            // This is a simple "Hello World" program
            Console.WriteLine("Hello, World!");
            
            // Variables and basic operations
            string name = "C# Developer";
            int year = 2024;
            
            Console.WriteLine($"Welcome {name} to C# programming in {year}!");
            
            // Wait for user input before closing
            Console.ReadLine();
        }
    }
}
Program Structure Breakdown:
  1. using statements: Import namespaces (like using System;)
  2. namespace declaration: Organizes related classes
  3. class declaration: Defines a class (blueprint for objects)
  4. Main method: Entry point of the application

Understanding .NET

The platform that powers C# applications

.NET Framework

The original .NET implementation for Windows development.

  • Windows-only
  • Mature and stable
  • Large ecosystem
  • Legacy applications
.NET Core / .NET 5+

Modern, cross-platform implementation of .NET.

  • Cross-platform (Windows, macOS, Linux)
  • High performance
  • Open source
  • Cloud-ready

Environment Information

Getting system information in C#

using System;

class EnvironmentInfo
{
    static void Main()
    {
        Console.WriteLine("=== C# Environment Information ===");
        
        // Display .NET version information
        Console.WriteLine($"Runtime Version: {Environment.Version}");
        Console.WriteLine($"Operating System: {Environment.OSVersion}");
        Console.WriteLine($"Machine Name: {Environment.MachineName}");
        Console.WriteLine($"User Name: {Environment.UserName}");
        Console.WriteLine($"Current Directory: {Environment.CurrentDirectory}");
        
        // Framework information
        Console.WriteLine($"64-bit OS: {Environment.Is64BitOperatingSystem}");
        Console.WriteLine($"64-bit Process: {Environment.Is64BitProcess}");
        Console.WriteLine($"Processor Count: {Environment.ProcessorCount}");
        
        Console.WriteLine("\nPress any key to exit...");
        Console.ReadKey();
    }
}
