Classes & Objects

Class is a blueprint. Object is an instance. Use constructors to initialize state. Properties provide controlled access to fields.

Example: Simple Person class
public class Person
{
    public string Name { get; set; }
    public int Age { get; private set; }

    public Person(string name, int age)
    {
        Name = name;
        Age = age;
    }

    public void Greet()
    {
        Console.WriteLine($"Hello, I am {Name} and I'm {Age} years old.");
    }
}

static void Main()
{
    var p = new Person("Alice", 30);
    p.Greet(); // Output: Hello, I am Alice and I'm 30 years old.
}

Explanation: Auto-properties, constructor initialization and a method that prints state.
